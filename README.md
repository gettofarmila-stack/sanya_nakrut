# Telegram бот для автонакрутки

Небольшой проект телеграм бота для работы с сервисами накрутки.  
Есть простая логика пополнения баланса и возможность подключить почти любой SMM API (v2) - достаточно указать ссылку и ключ в конфиге.

Проект делался как практический, поэтому основной упор на рабочую логику и удобную структуру кода, просьба не комментировать за не очень хорошее оформление.

---

## Что умеет

Бот позволяет:
- заказывать услуги накрутки прямо из Telegram
- простая платёжная логика
- подключаться к разным сервисам через API

---

## Установка

Клонируешь проект:
```bash
git clone https://github.com/gettofarmila-stack/sanya_nakrut.git
cd sanya_nakrut
```
Создаёшь виртуальное окружение:
```bash
python -m venv venv
```
Активируешь его:

Linux / MacOS:
```bash
source venv/bin/activate
```
Windows:
```bash
venv\Scripts\activate
```
Ставишь зависимости:
```bash
pip install -r requirements.txt
```
---

## Настройка

Создай файл config.py и укажи:
```python
my_token = "токен бота"
my_database = 'postgresql://postgres:1234@127.0.0.1:5432/nakrut' ссылка на БД
ADMINS = [айди админов] 
smm_key = 'АПИ ключ от смм сервиса'
smm_link = 'https://looksmm.ru/api/v2' ссылка на смм сервис такого типа
async_database = 'postgresql+asyncpg://postgres:1234@127.0.0.1:5432/nakrut' та же ссылка, но для асинхронного взаимодействия(извиняюсь, в процессе разработки учился асинхронному программировани.)
MY_ADDR = "0x67e21DCBFea64CE4a1b29c13CB2efeb63a5c44F4" адрес вашего BNB кошелька
```

## Как это в целом работает

Пользователь выбирает услугу → вводит ссылку → пополняет баланс → бот отправляет заказ в API → дальше можно отслеживать статус
